## Installation

### Backend
cd to backend folder and run
```bash
pip install -r requirements.txt
```

and run server with 
```bash
flask --app server.py run --reload 
```
### Frontend
cd to client folder and run
```bash
npm install --force
```

and run react with 
```bash
npm start
```
